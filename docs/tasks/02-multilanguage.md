# Aufgabe 2: Mehrsprachigkeit (i18n)

## Beschreibung

Füge Mehrsprachigkeit zur Applikation hinzu, sodass die UI in verschiedenen Sprachen verfügbar ist.

## Schritt 1: Deutsch und Englisch mit ngx-translate

- Integriere `ngx-translate` in das Angular-Frontend
- Erstelle Übersetzungsdateien für Deutsch und Englisch
- Ersetze alle hardcodierten Texte in der UI durch Übersetzungsschlüssel
- Füge einen Sprachwechsler in die UI ein (z.B. im Header)

## Schritt 2: Französisch hinzufügen

- Erstelle eine weitere Übersetzungsdatei für Französisch
- Erweitere den Sprachwechsler um die dritte Sprache
