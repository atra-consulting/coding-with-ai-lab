# Aufgabe 4: KI-Beziehungsanalyse (Claude API)

## Beschreibung

Implementiere eine KI-gestützte Analyse der Geschäftsbeziehung zu einer Firma oder Person mithilfe der Claude API.

## Anforderungen

- Füge einen Button **„Beziehungs-Summary"** in die Firmen- und Personen-Detailseite ein
- Beim Klick werden alle zugehörigen Aktivitäten, Verträge und Chancen gesammelt und an die Claude API übergeben
- Die Ausgabe enthält:
  - Eine Zusammenfassung der bisherigen Beziehung
  - Empfohlene nächste Schritte

## Lernziel

- Prompt Engineering: Wie formuliert man einen effektiven Prompt?
- Kontextaufbereitung: Wie bereitet man CRM-Daten strukturiert für ein LLM auf?
