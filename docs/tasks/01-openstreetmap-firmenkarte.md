# Aufgabe 1: OpenStreetMap – Firmen auf der Karte

## Beschreibung

Erweitere die Applikation um eine Kartenansicht mit OpenStreetMap, sodass Firmenstandorte visuell dargestellt werden.

## Schritt 1: Kartenansicht in der Firmen-Detailseite

- Integriere OpenStreetMap (z.B. via Leaflet) in die Firmen-Detailseite
- Zeige eine oder mehrere Adressen der Firma (Niederlassungen) als Marker auf der Karte an
- Die Karte soll sich automatisch so zoomen, dass alle Niederlassungen sichtbar sind

## Schritt 2: Routenberechnung zwischen Niederlassungen

- Füge eine Funktion hinzu, mit der die Route zwischen den Niederlassungen berechnet und auf der Karte angezeigt werden kann
- Nutze dafür einen Routing-Service (z.B. OSRM oder OpenRouteService)
