## Konventionen

- **Nummerierung**: Jede UXDR erhält eine fortlaufende dreistellige Nummer (`001`, `002`, …) im Dateinamen und Titel.
- **Status**: `Offen` → `Entschieden` → ggf. `Ersetzt durch UXDR-XXX`.
- **Bezug**: Jede UXDR verlinkt auf die PRD, aus der die Fragestellung hervorgeht.
- **Inhalt**: Vergleich von UX-Designansätzen mit Wireframes, Vor-/Nachteilen und Empfehlung.
- **Abgrenzung zu ADRs**: ADRs betreffen technische Architekturentscheidungen, UXDRs betreffen Interaktions- und Darstellungsentscheidungen.
